# :tada: Backwrap

## Working with Backwrap
- Fork the repository
- Clone the forked repository
- Run `npm install` to install the dependencies
- Add a **.env** file and set `PORT = 5000`
- Run `npm start` or `nodemon server` or `node server` to start the app
